# 디지털 명함 스킬 — 배포·설치 안내 (스터디 회원용)

## 이게 뭐예요?

`skill-digital-namecard` 폴더를 설치하면, 클로드에게 **"디지털 명함 만들어줘"** 한마디만 하면 됩니다.
복붙 프롬프트 없이 — 클로드가 알아서 인터뷰 7문 → 소개글 대필 → HTML 명함 제작까지 진행합니다. (myGPTs와 같은 개념. 차이: GPTs는 챗지피티 서버에 저장, 스킬은 내 컴퓨터 폴더에 저장 — 파일이라서 수정·복사·선물이 자유로움)

## 설치 (클로드 데스크톱 앱 / 클로드 코드)

1. `skill-digital-namecard` 폴더를 통째로 복사
2. 아래 위치에 붙여넣기:
   - 모든 프로젝트에서 쓰려면: `~/.claude/skills/skill-digital-namecard/`
   - 특정 폴더에서만 쓰려면: `그 폴더/.claude/skills/skill-digital-namecard/`
3. 클로드를 새로 열고 "디지털 명함 만들어줘" 라고 말하면 발동

## 배포 방법 (발행자용)

- 폴더를 zip으로 묶어 이메일·카톡·USB로 전달 → 회원이 위 위치에 압축 해제
- zip 만들기: `cd ~/Nerdnugget/byeolbyeol-newsletter && zip -r 디지털명함스킬.zip skill-digital-namecard`
- 뉴스레터 일반 회원에게는 스킬 대신 2호 본문의 복붙 프롬프트로 (설치 장벽 없음). 스킬은 스터디 참여 혜택으로.

## 고치고 싶을 때

`SKILL.md`가 전부입니다. 질문 7개를 바꾸고 싶으면 그 파일의 질문만 고치면 끝.
(가게 버전 질문으로 통째로 바꾼 사본을 `skill-shop-namecard`로 만들면 사장님용 스킬이 됩니다)
